# 工程简介
基于原“绿柚平台”所进行的第二代开发，在架构上采用完全的前后端分离模式，舍弃原有的Thymeleaf+SpringBoot的一体化结构，对项目整体架构做解构升级，添加认证授权模块，完善商城模块等。
